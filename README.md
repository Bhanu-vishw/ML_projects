# Data Science & Machine Learning Projects (Financial Applications)

This repository contains a collection of Data Science, Machine Learning, and Quantitative Finance projects developed as part of my coursework and independent experimentation. The projects focus on applying statistical learning, portfolio optimization, NLP, and financial modeling techniques to real-world financial datasets.

The repository demonstrates the use of supervised learning models, regularization techniques, feature engineering, portfolio optimization algorithms, and NLP pipelines for financial data analysis.


# Core Libraries Used

pandas, numpy, scikit-learn, xgboost, tensorflow, keras, spaCy, arch, riskparityportfolio, yfinance, pandas-datareader, matplotlib, seaborn, scipy, joblib


# Projects Overview

## Multi-Factor Long-Only Strategy

Designed a machine learning-based multi-factor stock selection strategy using factor ranking and predictive modeling.

Algorithms / Methods:
Random Forest, Logistic Regression, XGBoost (gradient boosted trees), Principal Component Analysis (PCA), feature preprocessing, hyperparameter tuning using GridSearchCV.

Libraries:
pandas, numpy, scikit-learn, xgboost, matplotlib, joblib

## Multi-Factor Long–Short Strategy

Developed a market-neutral factor-based trading strategy by ranking securities based on factor signals and constructing long and short portfolios.

Algorithms / Methods:
Cross-sectional factor ranking, portfolio sorting, long-short portfolio construction, risk-adjusted performance evaluation.

Libraries:
pandas, numpy, matplotlib, scikit-learn


## Portfolio Asset Allocation

Constructed optimized portfolios and analyzed risk-return tradeoffs using quantitative portfolio allocation methods.

Algorithms / Methods:
Risk parity portfolio construction, portfolio optimization, risk-return analysis, portfolio performance evaluation.

Libraries:
pandas, numpy, yfinance, pandas-datareader, riskparityportfolio, matplotlib


## K-Nearest Neighbors Classification

Implemented a predictive classification model using the K-Nearest Neighbors algorithm to classify observations based on feature similarity.

Algorithms / Methods:
K-Nearest Neighbors (KNN), feature scaling, train-test split evaluation, distance-based classification.

Libraries:
pandas, numpy, scikit-learn, seaborn, matplotlib.

## Volatility Forecasting

Developed financial volatility forecasting models using historical return data to estimate future market risk.

Algorithms / Methods:
GARCH models, time-series return modeling, regression-based forecasting, statistical volatility estimation.

Libraries:
pandas, numpy, scikit-learn, arch, yfinance, matplotlib, seaborn.

## Ridge and Lasso Regression

Explored regularization techniques to address multicollinearity and improve model generalization.

Algorithms / Methods:
Ridge regression (L2 regularization), Lasso regression (L1 regularization), coefficient shrinkage, feature selection.

Libraries:
pandas, numpy, scikit-learn


## Feature Selection using Hierarchical Clustering

Implemented correlation-based feature reduction using clustering techniques to remove redundant predictors.

Algorithms / Methods:
Hierarchical clustering, correlation matrix analysis, cluster-based feature selection, neural networks experimentation.

Libraries:
pandas, numpy, scipy, seaborn, matplotlib, tensorflow, keras


## Financial Sentiment Analysis

Built an NLP pipeline to classify sentiment in financial text datasets.

Algorithms / Methods:
TF-IDF vectorization, count vectorization, Logistic Regression, Multinomial Naive Bayes, K-Nearest Neighbors for text classification.

Libraries:
pandas, numpy, scikit-learn, spaCy
